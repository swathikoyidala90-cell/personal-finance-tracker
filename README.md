# Personal Finance Tracker

A web application to track your personal finances, manage transactions, and monitor spending. Built with Spring Boot and deployed on Render.

## Features

- Add, edit, and delete transactions
- Categorize expenses and income
- View all transactions in a user-friendly dashboard
- Track financial health over time
